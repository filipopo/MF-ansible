Mobile Forces multiplayer guide

This guide will explain on how to set the game up to be able to join the modded game servers with the following steps

  1. Download the updater script
  2. Join the server
  3. Links to zip archives of modded game files and addons
  4. Display the server in the Internet Servers list
  5. Modify MobileForces.ini by hand to display the server in the list

Download the updater script

Updater link:
https://www.mediafire.com/file/qyen4okc2a8qbau/Updater.zip/file

Download the zip archive and extract the Updater folder out of the zip archive into the main game folder. Consider backing up the game files since this updater will overwrite all files which will not match the server files. Launch script.bat file. The updating process should begin.

Join the server

Domain name and port number for the european server:
mf.nofisto.com:7790 (IP address and port number 49.13.114.72:7790)
IP address and port number for the canadian server:
45.74.103.79:10205
To connect, you might want to go to Multiplayer, Open location menus, put the domain name with the port number (or the IP address with the port number) and click Start.

Links to zip archives of modded game files and addons

Beware that some of the files in these zips might be outdated. If you download files with updater script then you don't need to download these except If you wish to see internet servers you might want to get the Update.zip (MobileForces.ini will be inside of it which you will want to put to the System folder) or If you wish to play with russian voiceovers you might want to get them from RUupdate.zip.

Display the servers in the Internet Servers list

In order to see the servers in the Internet Servers list (Multiplayer > Join Game > Internet Servers), you can replace your MobileForces.ini file located in MobileForces/System folder with the one in Update.zip.

Modify MobileForces.ini by hand to display the server in the list

If you don't want to overwrite MobileForces.ini, you can remove the lines under [RageBrowserAll]:

ListFactories[0]=UBrowser.UBrowserGSpyFact,MasterServerAddress=195.173.199.196,MasterServerTCPPort=28900,Region=0,GameName=mobileforces ListFactories[1]=UBrowser.UBrowserGSpyFact,MasterServerAddress=master.gamespy.com,MasterServerTCPPort=28900,Region=0,GameName=mobileforces

And replace them with these lines:

ListFactories[0]=UBrowser.UBrowserGSpyFact,MasterServerAddress=mf.nofisto.com,MasterServerTCPPort=28900,Region=0,GameName=mobileforces ListFactories[1]=UBrowser.UBrowserGSpyFact,MasterServerAddress=master.333networks.com,MasterServerTCPPort=28900,Region=0,GameName=mobileforces
